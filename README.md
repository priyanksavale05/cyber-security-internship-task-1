# cyber-security-internship-task-1

## Task Title
Understanding Cyber Security Basics & Attack Surface

## Task Description
This repository contains my submission for Task 1 of the Cyber Security Internship.
The objective of this task was to understand fundamental cyber security concepts such as
the CIA triad, types of cyber attackers, attack surfaces, OWASP Top 10 vulnerabilities,
and data flow in applications.

## Tools & Resources Used
- Web Browser (Google Chrome)
- OWASP Official Website
- Cyber security articles and blogs for self-research
- Notes prepared during research

## Work Done
- Studied basic cyber security principles including Confidentiality, Integrity, and Availability (CIA Triad)
- Researched different types of cyber attackers and their motivations
- Understood the concept of attack surface and common attack points
- Reviewed OWASP Top 10 web application security risks
- Analyzed data flow in typical applications and identified possible attack locations
- Compiled all findings into a structured and professional PDF document
